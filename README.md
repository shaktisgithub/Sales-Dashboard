# Superstore Sales Dashboard

![visualization](https://github.com/user-attachments/assets/d1e3d953-7a9f-4890-b33a-8c93ed01c722)


## Overview

This repository contains a **Sales Dashboard** created using **Power BI** to provide insights into the sales performance of a superstore. The dashboard is built with visualizations that focus on various metrics such as sales, profit, quantity, and delivery times, segmented by regions, payment modes, and product categories.

## Dashboard Highlights

The Sales Dashboard includes the following key visualizations:

1. **Sum of Sales by Payment Mode**: Shows the breakdown of sales through different payment methods (Cards, COD, Online).
2. **Sum of Sales by Segment**: Displays sales categorized by different customer segments (Home Office, Consumer, Corporate).
3. **Sales and Profit by Region**: Geographical map highlighting the total sales and profit per state.
4. **Sales Trends by Month and Year**: Line charts showing the sales and profit trends for the years 2019 and 2020.
5. **Sum of Sales by Category and Sub-Category**: Visualizes the sales distribution across various product categories and sub-categories.
6. **Sum of Profit by Month and Year**: Tracks profit trends on a monthly basis.
7. **Sum of Sales by Ship Mode**: Highlights the shipping methods used for the orders.

## Dataset

The dashboard is created using a **Superstore Sales Dataset**. The dataset contains the following fields:

- **Order ID**
- **Region**
- **Category**
- **Sub-Category**
- **Sales**
- **Quantity**
- **Profit**
- **Ship Mode**
- **Payment Mode**
- **State**
- **Segment**
- **Year** (2019 and 2020)

The dataset has been used to explore and analyze various aspects of business performance and identify trends across regions, categories, and other business parameters.

## Tools Used

- **Power BI**: The primary tool used for creating the dashboard and visualizations.
- **Microsoft Excel/CSV**: The dataset is available in CSV format and was cleaned and prepared using Excel.

## How to Use

1. Clone this repository to your local machine.
2. Open the dataset file (`SuperStore_Sales_Dataset.csv`).
3. The dashboard visualizes the key business metrics, allowing you to analyze different sales dimensions, such as:
   - Region-based sales and profits
   - Category-based performance
   - Monthly sales and profit trends
